Russian:

Этот проект - интерпретатор языка программирования LOGO.

Проект был разработан в феврале 1990 года. В то время я учился на 3 курсе Рижского авиационного университета (факультет вычислительной техники) и самостоятельно изучал язык программирования PASCAL. Я услышал о концепции языка LOGO для начального обучения программированию. И решил попробовать реализовать упрощенный вариант этого языка на Turbo Pascal. И мне это удалось.

Основной файл проекта: LOGO.PAS
Компилируется Turbo Pascal 4.0 (или выше)
Кодировка кириллицы OEM 866

English:

This project - the LOGO programming language interpreter.

The project was developed in February 1990. At that time I was a 3 course Riga Aviation University (Faculty of Computer Engineering) and taught himself programming language PASCAL. I heard about the concept of LOGO language for the initial training of programming. And I decided to try to implement a simplified version of the language LOGO to Turbo Pascal. And I succeeded.

The main file of the project: LOGO.PAS
Compiled Turbo Pascal 4.0 (or higher)
Encoding Cyrillic OEM 866
